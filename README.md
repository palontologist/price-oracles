# Price Oracles

Wheat and Maize Floor Price Oracles - A Next.js 15 application for fetching real-time commodity prices from multiple data sources.

## Coming Soon

This repository is being set up with:
- Next.js 15 with App Router
- TypeScript
- Tailwind CSS
- Drizzle ORM with Turso/libSQL
- Multi-source price oracle integration (Tridge, World Bank, Alpha Vantage)

## Features

- Real-time wheat and maize price fetching
- Automatic fallback through multiple data sources
- Historical price storage
- RESTful API endpoints
- Frontend dashboard
